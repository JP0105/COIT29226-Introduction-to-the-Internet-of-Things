## Task 2: Devices, Protocols, and Services in ThingsBoard

After exploring the ThingsBoard tutorial and the IoT system example, I identified devices, protocols, and services used in the platform. These can be matched with the IoT architecture models as shown below.

| Category | Examples in ThingsBoard | 3-Layer Architecture Model | 5-Layer Architecture Model |
|----------|--------------------------|----------------------------|----------------------------|
| Devices | Sensors, smart meters, GPS trackers, IoT nodes, connected machines | Perception Layer | Perception Layer |
| Protocols | MQTT, HTTP, CoAP | Network Layer | Transport Layer |
| Services | Device management, data collection, rule engine, alarm system, dashboards, data visualization, analytics | Application Layer | Processing Layer / Application Layer |

### Explanation

The devices are the physical objects that collect information from the environment. For example, sensors can measure temperature, humidity, or movement. These belong to the perception layer because they are responsible for sensing and collecting data.

The protocols such as MQTT, HTTP, and CoAP are used to transfer data between devices and the platform. In the 3-layer model, they are part of the network layer because they help communication happen. In the 5-layer model, they belong to the transport layer because they carry data from one part of the system to another.

The services provided by ThingsBoard include dashboards, rule engine, analytics, alarms, and device management. These help process, monitor, and display the collected data. In the 3-layer model, these are included in the application layer. In the 5-layer model, they are mainly part of the processing and application layers.

## Task 3: IoT Applications I Use in Daily Life

There are many IoT applications that I use in daily life. Some of them are:

- Smartphone with Google Maps
- Smartwatch or fitness band
- Smart TV
- Food delivery apps with live tracking
- Online shopping parcel tracking
- Car GPS and parking sensors
- CCTV cameras connected to mobile apps

### How IoT has made life easier

IoT has made everyday life easier in many ways. One example is Google Maps, which gives real-time traffic updates and helps save time while travelling. Food delivery apps also use IoT and location tracking so I can see where the rider is and know exactly when the order will arrive.

A smartwatch or fitness band is another useful example because it tracks steps, heart rate, calories, and sleep. This helps in maintaining health and daily routine. Smart TVs also make life easier by connecting entertainment services through the internet and giving personalized recommendations.

Parcel tracking is also very convenient because it gives live updates about where the package is. CCTV cameras connected to a phone improve security because they allow remote monitoring. Overall, IoT saves time, improves convenience, and gives more control in daily life.
